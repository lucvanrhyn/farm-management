Where should the analytics live?

The admin currently has 5 tabs. We need to decide where graphs and analytics fit in.

A

New "Grafieke" tab in admin nav

A dedicated analytics page — /admin/grafieke — with all 4+ charts on one scrollable dashboard. Clean separation from the data tables. Adds a 6th tab to the sidebar.

Pros

  • Focused analytics experience
  • Easy to expand later
  • Doesn't clutter existing pages

Cons

  • One more tab to click to
B

Expand the existing "Oorsig" (Overview) page

The current /admin overview already has 4 stat cards + 2 panels. Add charts directly below the stats — analytics become part of the landing page when you enter admin.

Pros

  • Fewer tabs to navigate
  • Analytics front and center on login

Cons

  • Page gets long
  • Harder to find specific data
C

Replace "Oorsig" with a full analytics dashboard

Remove the current overview and replace it with a proper analytics dashboard — stat cards at top, charts below. The /admin route becomes the analytics home.

Pros

  • No extra tabs
  • Analytics IS the overview
  • Clean slate for richer design

Cons

  • Replaces existing overview work
