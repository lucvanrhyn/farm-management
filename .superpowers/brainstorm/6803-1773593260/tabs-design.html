Grafieke — Full tab design

What lives in each tab. Both use Recharts, server-fetched, 30-day window.

📍 Kampe Tab — 6 items

1. Camp Condition Trend
Line chart — last 30 days
Data: camp_condition observations grouped by day, avg grazing quality score (Goed=4, Redelik=3, Swak=2, Oorblaai=1). One line per camp (or avg all camps).
2. Health Issues by Camp
Bar chart — last 30 days
Data: Count of health_issue observations grouped by campId. Bars coloured red/orange by severity. X-axis: 19 camps.
3. Headcount per Camp
Stacked bar — current snapshot
Data: Active animals grouped by currentCamp + category (Cow/Bull/Heifer/Calf/Ox). Stacked colours per category. X-axis: 19 camps.
4. Stocking Rate / Grazing Pressure
Bar with target line — current snapshot
Data: Animals per hectare per camp (headcount ÷ camp size_hectares). Reference line at optimal density (e.g. 1 LSU/ha). Bars red if over threshold.
5. Inspection Heatmap
Grid heatmap — last 30 days
Data: Count of camp_check + camp_condition per camp per day. Grid: camps (Y) × days (X). Darker = more inspections. White = no visit.
6. Animal Movement Flow
Table — last 30 days
Data: animal_movement observations. Table: Date | Animal ID | From camp → To camp | Logged by. Sorted newest first. Simpler than a Sankey for now.

🐄 Diere Tab — 3 items

7. Calving Rate Trend
Bar chart — last 12 months
Data: reproduction observations where details.event includes "calving", grouped by month. Shows calf count per month.
8. Deaths & Sales Over Time
Grouped bar — last 12 months
Data: Animals where status changed to Deceased/Sold, grouped by month. Two bars per month: Deaths (red) + Sales (blue).
9. Treatment Withdrawal Tracker
Table — active only
Data: treatment observations with withdrawal_days > 0 where observedAt + withdrawal_days > today. Shows: Animal ID | Drug | Days remaining | Camp. Red if < 3 days.

All charts styled in the existing admin stone/brown palette. Recharts tooltips in Afrikaans.
